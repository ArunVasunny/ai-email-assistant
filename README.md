# AI Smart Email Assistant Extension

This is a browser extension that generates AI-powered email replies directly inside Gmail. It reads the email content and helps draft context-aware responses, with optional tone customization before sending.

This project was built to explore how AI can be integrated into real-world workflows like email communication using browser extensions, backend APIs, and LLMs.


##  Features

-  Generate AI-based email replies inside Gmail  
-  Context-aware response generation  
-  Optional tone selection (e.g., professional, friendly)  
-  Edit and review before sending  
-  Seamless integration with Gmail interface  


##  Technologies Used

- Spring Boot  
- Spring AI  
- Gemini API  
- JavaScript (Chrome Extension)
